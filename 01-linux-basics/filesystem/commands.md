Prints the current working directory.
```bash
pwd
```

List the files in the current working diretory
```bash
ls
```

List the files including hidden files
```bash
ls -a
```

List the files with detailed information
```bash
ls -l
```

Show detailed information for all files, including hidden ones
```bash
ls -la
```

List the files with Human-readable file sizes (KB, MB, GB)
```bash
ls -lh
```

Change the directory
```bash
cd
```

Moves up one level into the parent folder

```bash
cd ..
```
Takes you directly back to your home directory (your user profile folder)

```bash
cd or cd ~ 
```

Create new folders(directories)
```bash
mkdir
```

Create a new empty file or update the last modified timestamp of an existing file
```bash
touch
```

Copy files or directories from one location to another
```bash
cp
```

Cut and paste files or directories from one location to another, or rename it
```bash
mv
```

Remove (delete) files or directories
```bash
rm
```
